# emoji-json

Just a list of `[emoji, alias, unicode]` tuples.


## Installation

```
npm i emoji-json
```


## Usage

```js
const {list} = require('emoji-json');

console.log(list[0]);
console.log(list[1]);
console.log(list[2]);
```
